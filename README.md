# Indonesians in Dubai — SITREP

Real-time situation reports and emergency information for Indonesian nationals in Dubai and the UAE during the Middle East crisis.

## What Is This?

A [Mintlify](https://mintlify.com)-powered documentation site providing:

- **Structured SITREPs** with verified intelligence from multiple OSINT sources
- **DEFCON-style threat levels** (4 = Elevated → 1 = Evacuate)
- **Emergency contacts** for Indonesian embassies and UAE services
- **Exit route planning** including pet logistics for families with animals
- **Go-bag checklists** for emergency preparedness

## Development

```bash
npm i -g mintlify
mintlify dev
```

## Disclaimer

This site is maintained by volunteers and AI-assisted monitoring. It is **not affiliated** with the Indonesian government or any official entity. Always verify critical decisions with official sources.
