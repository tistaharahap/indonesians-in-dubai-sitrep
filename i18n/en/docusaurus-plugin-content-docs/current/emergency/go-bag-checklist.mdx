:::info
At **DEFCON 3 or higher**, have these bags packed and ready at the door. At **DEFCON 2**, be ready to move within **30 minutes**.
:::

## Documents

- [ ] **Passports** (all family members — Indonesian + any other)
- [ ] **UAE residence visa** cards/documents
- [ ] **Pet vaccination records** (rabies certificates — required for Oman entry)
- [ ] **Pet microchip documentation**
- [ ] **Travel insurance** documents
- [ ] **Copies of all documents** (physical + digital on phone/cloud)
- [ ] **Peduli WNI registration** confirmation

## Cash & Finances

- [ ] **USD cash** — $1,000+ recommended (universally accepted)
- [ ] **AED cash** — 2,000+ AED for local expenses
- [ ] **Multiple debit/credit cards** from different banks
- [ ] **IDR** — some Indonesian Rupiah if available

## Electronics

- [ ] **Phone chargers** + portable power banks (fully charged)
- [ ] **Universal adapter**
- [ ] **Offline maps** downloaded (Google Maps: UAE, Oman)
- [ ] **Important contacts** saved offline

## Medical & Hygiene

- [ ] **Medications** — at least 2 weeks supply for all family members
- [ ] **First aid kit**
- [ ] **Personal hygiene essentials**
- [ ] **Prescription copies**

## Supplies

- [ ] **Water bottles** (2L per person minimum)
- [ ] **Non-perishable food** (energy bars, nuts, dried fruit)
- [ ] **Change of clothes** for each person
- [ ] **Light blankets / emergency blankets**
- [ ] **Flashlight** + spare batteries

## Pet Supplies 🐕

- [ ] **Pet carriers** (airline-approved if possible)
- [ ] **Dog food** — 3+ days supply
- [ ] **Water bowl** (collapsible)
- [ ] **Leashes and harnesses**
- [ ] **Pet medications** if any
- [ ] **Comfort items** (familiar toy/blanket)
- [ ] **Waste bags**

## Vehicle Prep

- [ ] **Fuel tank FULL**
- [ ] **Spare jerry cans** filled if available
- [ ] **Extra water** in the car (5L+)
- [ ] **Tire pressure** checked
- [ ] **Phone mount** + car charger
- [ ] **Physical map** of UAE/Oman as backup
