:::info
Pada **DEFCON 3 atau lebih tinggi**, siapkan tas ini dan letakkan di depan pintu. Pada **DEFCON 2**, siap bergerak dalam **30 menit**.
:::

## Dokumen

- [ ] **Paspor** (semua anggota keluarga — Indonesia + lainnya)
- [ ] **Kartu/dokumen visa tinggal UEA**
- [ ] **Catatan vaksinasi hewan peliharaan** (sertifikat rabies — diperlukan untuk masuk Oman)
- [ ] **Dokumentasi microchip hewan peliharaan**
- [ ] **Dokumen asuransi perjalanan**
- [ ] **Salinan semua dokumen** (fisik + digital di ponsel/cloud)
- [ ] **Konfirmasi pendaftaran Peduli WNI**

## Uang Tunai & Keuangan

- [ ] **Uang tunai USD** — $1.000+ disarankan (diterima secara universal)
- [ ] **Uang tunai AED** — 2.000+ AED untuk pengeluaran lokal
- [ ] **Beberapa kartu debit/kredit** dari bank berbeda
- [ ] **IDR** — beberapa Rupiah Indonesia jika tersedia

## Elektronik

- [ ] **Charger ponsel** + power bank portabel (terisi penuh)
- [ ] **Adaptor universal**
- [ ] **Peta offline** sudah diunduh (Google Maps: UEA, Oman)
- [ ] **Kontak penting** tersimpan secara offline

## Medis & Kebersihan

- [ ] **Obat-obatan** — minimal persediaan 2 minggu untuk semua anggota keluarga
- [ ] **Kotak P3K**
- [ ] **Perlengkapan kebersihan pribadi**
- [ ] **Salinan resep**

## Persediaan

- [ ] **Botol air** (minimal 2L per orang)
- [ ] **Makanan tahan lama** (energy bar, kacang, buah kering)
- [ ] **Pakaian ganti** untuk setiap orang
- [ ] **Selimut ringan / selimut darurat**
- [ ] **Senter** + baterai cadangan

## Perlengkapan Hewan Peliharaan 🐕

- [ ] **Tas hewan peliharaan** (sesuai standar maskapai jika memungkinkan)
- [ ] **Makanan anjing** — persediaan 3+ hari
- [ ] **Mangkuk air** (bisa dilipat)
- [ ] **Tali pengikat dan harness**
- [ ] **Obat hewan peliharaan** jika ada
- [ ] **Barang kenyamanan** (mainan/selimut yang familiar)
- [ ] **Kantong sampah**

## Persiapan Kendaraan

- [ ] **Tangki bahan bakar PENUH**
- [ ] **Jerigen cadangan** terisi jika tersedia
- [ ] **Air tambahan** di mobil (5L+)
- [ ] **Tekanan ban** diperiksa
- [ ] **Dudukan ponsel** + charger mobil
- [ ] **Peta fisik** UEA/Oman sebagai cadangan
